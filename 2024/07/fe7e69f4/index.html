线性代数

我们定义 $\hat{\imath}_i$ 为只有第 $i$ 个维度为 $1$ 、其余维度都是 $0$ 的向量。特别地，在维度不超过 $3$ 时，用 $\hat{\imath}$ 代表 $\hat{\imath}_1$ ，$\hat{\jmath\,}$ 代表 $\hat{\imath}_2$ ，$\hat{k}$ 代表 $\hat{\imath}_3$ 。

线性变换 $\boldsymbol{v}\mapsto L(\boldsymbol{v})$ 是满足以下条件的变换（变换指到自身的映射）：

  1. 直线变换后仍是直线。
  2. 原点变换后位置不变。

若不满足第二点，可称为“仿射变换”。
应将线性变换看作“保持网格线平行且等距分布”的变换。

矩阵（方阵）可以看作线性变换。
具体来说，对于代表线性变换 $L$ 的一个 $n$ 阶方阵：

把它看作 $n$ 个列向量：

将 $\hat{\imath}’_i$ 看作 $\hat{\imath}_i$ 线性变换后所处的位置，即 $\hat{\imath}’_i=L(\hat{\imath}_i)$ 。

将一个向量 $\boldsymbol{v}=\begin{bmatrix}a1\\a_2\\\vdots\\a_n\end{bmatrix}$ 看作 $\sum\limits{i=1}^na_1\hat{\imath}_i$ ，那么容易看出

例：二维矩阵 $\begin{bmatrix}0&-1\\1&0\end{bmatrix}$ 表示的线性变换是将单位向量 $\hat{\imath}=\begin{bmatrix}1\\0\end{bmatrix}$ 变成 $\begin{bmatrix}0\\1\end{bmatrix}$ ，将单位向量 $\hat{\jmath\,}=\begin{bmatrix}0\\1\end{bmatrix}$ 变成 $\begin{bmatrix}-1\\0\end{bmatrix}$ ，即逆时针旋转九十度。
二维矩阵 $\begin{bmatrix}1&1\\0&1\end{bmatrix}$ 表示的线性变换是单位向量 $\hat{\imath}$ 不变，将单位向量 $\hat{\jmath\,}$ 变成 $\begin{bmatrix}1\\1\end{bmatrix}$ ，即把网格上所有竖线推斜至与 $x$ 轴夹角为 $45^\circ$ 而横线保持不变，称作“剪切”。


这是向量 $\begin{bmatrix}1\\2\end{bmatrix}$ 进行逆时针旋转九十度操作后的结果。

矩阵乘法可以看作多个线性变换的复合。
具体来说，对于 $M_3=M_2M_1$ ，可以看作先进行线性变换 $M_1$ ，再进行线性变换 $M_2$ ，等价于直接进行线性变换 $M_3$ 。具体操作是把 $\hat{\imath}_i$ 依次进行 $M_1,M_2$ 所代表的线性变换。

{ % note % }

为什么要从右向左？因为向量进行矩阵代表的线性变换需要矩阵乘以向量，依次进行。

{ % endnote % }

例：对于矩阵 $M_1=\begin{bmatrix}0&-1\\1&0\end{bmatrix}$ 和 $M_2=\begin{bmatrix}1&1\\0&1\end{bmatrix}$ ，矩阵乘法 $M_3=M_2M_1$ 表示的是先进行剪切，再进行旋转。容易看到 $M_3=\begin{bmatrix}1&-1\\1&0\end{bmatrix}$ 。
