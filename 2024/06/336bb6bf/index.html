dp 套 dp

发表于2024-06-29|更新于2024-06-30
|阅读量:

前言

这是一个令人头秃的鬼东西。
这与 ddp（动态动规）有很大区别。

介绍

问题

给定一个 dp 问题和一个答案，求有多少种输入可以让该 dp 获得该答案。

思路

众所周知，dp 可以看作 DFA(OI-Wiki) 。于是我们在给定的 dp 外面再套一层 dp，外层 dp 的状态中有一维是内层 dp 对应的 DFA 的结点。可以看作在 DAG 上计数（路径数）。若将内层 dp 状态设为 f⋯f_{\cdots}f⋯​ ，则外层 dp 状态即为 F⋯ ,XF_{\cdots,X}F⋯,X​ ，其中 X={f⋆}X=\{f_{\star}\}X={f⋆​} 。

然而一般来说向 dp 状态里面塞 DFA 十分过分，这时我们可以考虑状压。一般思路是想办法将 XXX 压扁再通过一系列技巧（如差分等）变成 010101 串。

例题

BZOJ 3864 / 黑暗爆炸 3864 / 洛谷 P10614

题意：给定 n,Tn,Tn,T，字符集 Σ={A,G,C,T}\Sigma=\{\tt{A,G,C,T}\}Σ={A,G,C,T} ，对于每个 i∈[0,∣T∣]∩Zi\in[0,|T|]\cap\mathbb Zi∈[0,∣T∣]∩Z ，询问有多少个长度为 nnn 的串 SSS 满足 LCS⁡(S,T)=i\operatorname{LCS}(S,T)=iLCS(S,T)=i 。（变量名与原题不符）

设 fi,jf_{i,j}fi,j​ 表示 LCS⁡\operatorname{LCS}LCS 的长度，易得转移

fi,j=max⁡{fi−1,j,fi,j−1,fi−1,j−1+[Si=Tj]}f_{i,j}=\max\{f_{i-1,j},f_{i,j-1},f_{i-1,j-1}+[S_i=T_j]\} fi,j​=max{fi−1,j​,fi,j−1​,fi−1,j−1​+[Si​=Tj​]}

再设 Fi,XF_{i,X}Fi,X​ 表示长度为 iii ，fff 数组为 XXX 的方案数。这样所有满足 XXX 中 fn,∣T∣=if_{n,|T|}=ifn,∣T∣​=i 的 Fn,XF_{n,X}Fn,X​ 的和即为所求。

首先注意到 fi,jf_{i,j}fi,j​ 只和 fi−1,⋆f_{i-1,\star}fi−1,⋆​ 有关，于是我们可以只记录 FFF 添加字符后最后一行 fff ，这样 XXX 从 f⋆,⋆f_{\star,\star}f⋆,⋆​ 变成了 fi,⋆f_{i,\star}fi,⋆​ ；接着发现 fi,j−fi,j−1∈{0,1}f_{i,j}-f_{i,j-1}\in\{0,1\}fi,j​−fi,j−1​∈{0,1} ，于是差分后状压即可。
